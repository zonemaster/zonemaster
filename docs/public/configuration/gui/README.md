# GUI Configuration

This section contains information about configuration of the GUI component.

* [Configuring Zonemaster GUI using "config.json"] is about *run-time
  configuration*, and that is currently limited to creating or updating a message
  banner.
* [Configuring Zonemaster GUI using "config.ts"] is about simple but *build-time
  configurations*.
* [Configuring Zonemaster GUI using "tsconfig.json"] is about advanced
  *build-time configurations* that can change the look-and-feel of
  Zonemaster-GUI.
* [Building a custom Zonemaster-GUI] has instructions for building a customized
  GUI.
